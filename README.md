# Simple-Quiz — Quiz Web App

**ProjectUpdated** is a simple, lightweight web application for creating and taking quizzes.  
It uses plain HTML, CSS, and JavaScript, with no backend required. Quizzes are defined in JSON format and can be easily customized.

---

## 🌟 Features

- Multiple quizzes support (`quiz1`, `quiz2`, `quiz3`)  
- Dynamic quiz display with question menu  
- Color-coded answers: green for correct, red for incorrect  
- Score calculation and result page after quiz completion  
- Navigation button to return to main page  
- Responsive design and clean UI  

---

## 🚀 How to Run

1. Clone or download the repository  
2. Open `index.html` in a browser (Chrome, Firefox, Edge — modern browsers recommended)  
3. Select a quiz and answer the questions  
4. After completing the quiz, your score will appear on the result page  

5. Click **Return to Main Page** to choose another quiz
